# Real Time Chat Application
A real-time chat application where users can login, chat with other users and send images.

## Tools 
- **React** and **Context API** (for state management) 
- **Firebase**
>  - Firestore database, 
>  - Real-time database
>  - Firebase Storage
>  - Firebase Authentication
